How It Works

  • Sender: Generates 3 QR codes (Cyan, Magenta, Yellow channels) and composites them. Adds 8 calibration patches with known CMY values around the border.
  • Receiver: Samples the calibration patches to measure how the camera distorts colors. Computes a 3x3 correction matrix via least squares. Applies this matrix to recover true CMY values before thresholding.
  • Key insight: Camera sensors have cross-channel interference (red bleeds into green, etc.). The correction matrix compensates for this device-specific distortion.
