Palette-RGB Sender

8-Color Palette Encoding: Each QR module is assigned a color from a fixed 8-color palette.
Three data channels (R, G, B) combine into a 3-bit index (0-7) that selects the palette color.
Receiver uses nearest-neighbor classification to map scanned colors back to palette indices.

Palette Classification Approach

  • Encoding: 3-bit index (0-7) selects color from fixed palette
  • Decoding: Nearest-neighbor finds closest palette color
  • Holistic classification - treats each module as single color
  • Robust to color drift if palette colors remain distinguishable
  • Alternative: Channel separation decodes R, G, B bits independently
